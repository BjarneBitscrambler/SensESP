Detailed Description

Classes that "WantsEnable" will have their enable() method call automatically at startup when the SensESP app itself is enabled. The optional priority allows for certain classes to be initialized first. A lower priority (e.g. zero) will be enabled before a higher one (e.g. 10). The maximum priority is 10.
