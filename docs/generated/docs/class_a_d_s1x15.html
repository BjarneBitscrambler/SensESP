Detailed Description

template<class T_Ada_1x15>
class ADS1x15< T_Ada_1x15 >

ADS1x15 represents an ADS1015 or ADS1115 Analog to Digital Converter.

In main.cpp, use one of these typedefs: ADS1015 or ADS1115

Parameters
addrThe address through which the chip reports its readings. The default is 0x48, but can be changed on most chips - see the datasheet.
gainOne of the adsGain_t values that determines the precision of the output. See https://github.com/adafruit/Adafruit_ADS1X15/blob/master/Adafruit_ADS1015.h
config_pathNot used for this Sensor, so you can omit it when you create an instance of this Class in main.cpp.
