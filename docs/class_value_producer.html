Detailed Description

template<typename T>
class ValueProducer< T >

A ValueProducer<> is any device or piece of code that outputs a value for consumption elsewhere. They are Observable, allowing code to be notified whenever a new value is available. They can be connected directly to ValueConsumers of the same type using the connectTo() method.
